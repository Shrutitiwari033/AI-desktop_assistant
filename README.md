# AI-desktop_assistant
# AI Desktop Assistant using Google Generative AI

Welcome to **AI Desktop Assistant**, a smart voice-activated assistant for Windows built using Python and powered by Google Generative AI (Gemini). This project listens to your voice commands, chats intelligently, opens websites and applications, plays music, and tells the time — all hands-free!

---

## 🔍 Features

- 🎙️ **Voice Recognition** with `speech_recognition`
- 🧠 **Smart Conversations** using Google GenAI (`gemini-pro`)
- 🌐 **Open Websites** like YouTube, Wikipedia, Google via voice
- 🎵 **Play Music** from local storage
- ⏰ **Tell Current Time**
- 📷 **Open Camera App** on Windows
- 📁 **Save AI Responses** for future reference
- 💬 **Reset Chat Memory** on demand
